Initial bracket size should be a square of 2. IE. 2, 4, 8, 16, 32, etc.

```js
function getBracketSize(numPlayers){
    for(let exp = 1; ;exp++){
        if(Math.pow(2, exp) >= numPlayers) return Math.pow(2, exp);
    }

}
```
